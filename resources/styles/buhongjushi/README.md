# 不红居士风格资料

本目录保存 B 站“红楼梦文本探究”视频形成的风格语料。

## 内容

- `transcripts/`: 视频转录文本、字幕和结构化 JSON。
- `metadata/`: 合集清单、manifest、运行状态和来源说明。
- `audio/`、`videos/`: 本地工作缓存，默认不提交。

## 使用边界

- `不红居士` 是后续交互机器人的风格名，来源于这组视频的文本细读和细节拆解方式。
- 风格资料用于控制表达方式、分析路径和用户偏好的对话口吻。
- 风格资料不能作为原文校订的最高证据；原文、注释、批语和图片字形仍以基础资料为准。
- 转录文本中的讲解者自称按原文保留，例如 `不红君` 不因风格名 `不红居士` 而替换。
