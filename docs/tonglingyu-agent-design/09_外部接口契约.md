# 09 外部接口契约

## 目标

外部接口用于连接 Open WebUI、管理后台和系统运维。本文档只定义能力和语义，不绑定具体技术协议、路径、编程语言或 SDK。

## 外部接口分组

| 接口组 | 使用方 | 目的 |
|---|---|---|
| 模型发现接口 | Open WebUI | 发现可用模型 |
| 聊天请求接口 | Open WebUI | 发送用户问题并接收回答 |
| 健康检查接口 | 运维系统 | 判断 Gateway 是否可用 |
| 追踪查询接口 | 管理员 | 查询请求链路和审校状态 |
| 证据包查询接口 | 管理员或审核人员 | 查看回答依赖证据 |
| 配置查看接口 | 管理员 | 查看策略状态，不暴露密钥 |
| 评测触发接口 | 测试或治理人员 | 运行标准问题集验证 |

## 模型发现接口

### 目的

让 Open WebUI 只发现一个可见模型：

> 通灵玉

### 输入语义

1. 调用方身份；
2. 可选的租户或用户组上下文。

### 输出语义

1. 可见模型标识；
2. 模型显示名称；
3. 模型说明；
4. 是否可用；
5. 不返回内部 Agent 列表。

### 验收标准

1. 普通用户只能看到一个模型；
2. 不出现 `honglou-main`、`honglou-text`、`honglou-commentary`、`honglou-reviewer`；
3. 未授权调用者不能发现模型。

## 聊天请求接口

### 目的

接收用户问题，返回最终回答。

### 输入语义

1. 可见模型标识；
2. 用户消息；
3. 会话标识；
4. 调用方身份；
5. 可选的流式或非流式偏好；
6. 不接受用户指定内部 Agent；
7. 不接受用户关闭审校。

### 输出语义

1. 最终回答；
2. 回答状态；
3. 可选的证据链编号；
4. 可选的简要来源说明；
5. 错误或降级说明。

### 验收标准

1. 请求能进入 Gateway；
2. Gateway 能映射到内部流程；
3. 最终回答来自审校后结果；
4. 错误时不泄露内部配置；
5. 用户不能通过输入控制内部工具权限。

## 健康检查接口

### 目的

判断系统是否可以接受请求。

### 输出语义

1. Gateway 状态；
2. Hermes Runtime 连接状态；
3. 知识服务状态；
4. 审校能力状态；
5. 最近错误摘要。

### 验收标准

1. 能区分 Gateway 自身故障和下游故障；
2. 下游故障时可触发降级；
3. 不泄露内部密钥或详细栈信息。

## 追踪查询接口

### 目的

让管理员查询某次回答的内部处理链路。

### 输入语义

1. 追踪编号；
2. 管理员身份；
3. 查询范围。

### 输出语义

1. 请求状态；
2. Agent 调用顺序；
3. 每步状态；
4. 审校结果；
5. 错误类别；
6. 不暴露系统提示词和密钥。

## 证据包查询接口

### 目的

查看某次回答依赖的证据卡片和结论声明。

### 输入语义

1. 证据包编号；
2. 管理员或审核人员身份；
3. 查询原因。

### 输出语义

1. 证据卡片列表；
2. 证据来源；
3. 支持范围；
4. 不支持范围；
5. 结论声明；
6. 证据等级；
7. 审校状态。

## 评测触发接口

### 目的

运行标准问题集，验证检索、回答和审校质量。

### 输入语义

1. 评测集名称；
2. 评测范围；
3. 触发人；
4. 评测版本。

### 输出语义

1. 评测结果摘要；
2. 失败问题列表；
3. 指标变化；
4. 是否达到上线门槛。

## 外部接口负面要求

外部接口不得：

1. 暴露内部 Agent；
2. 暴露工具白名单；
3. 暴露系统提示词；
4. 允许用户关闭审校；
5. 允许用户指定内部 profile；
6. 返回内部异常栈；
7. 返回未脱敏的审计日志。
